# 洛克王国 · 异色保底追踪工具

> 精准追踪三池保底进度。

---

## 功能概览

| 功能 | 说明 |
|------|------|
| **随机池追踪** | 记录每次随机捕捉的精灵名称与计数，支持增 / 减 / 重置 |
| **家族池追踪** | 按精灵家族分组记录保底进度，支持搜索过滤、独立重置 |
| **属性池追踪** | 18 种属性各自独立计数 |
| **操作日志** | 完整记录每次操作的时间戳、池子类型与计数快照，支持分类过滤 |
| **多存档管理** | 支持新建 / 切换 / 删除存档 |
| **实时持久化** | 每次操作后自动写入 JSON |

## 快速开始

**提示：如果您下载的是 Releases 页面已打包的程序** > 您**无需**安装 Python 环境和依赖。直接解压下载的压缩包，双击运行目录下的 `RocoCapture-x.x.x-Win-x64.exe` 即可启动。启动后将会在程序位置创建一个`saves`文件夹，用于保存存档。

### 环境要求

- Python 3.10 或以上
- Windows 10 / 11（DPI 感知功能依赖 Windows API）

### 安装依赖

项目使用虚拟环境隔离依赖，推荐以下方式：

```powershell
# 1. 克隆 / 解压项目后进入目录
cd RocoCaptureV2

# 2. 创建虚拟环境
python -m venv .venv

# 3. 安装依赖
.venv\Scripts\pip install -r requirements.txt
```

### 启动程序

```powershell
.venv\Scripts\python main.py
```

## 使用说明

### 存档管理

- 首次启动自动创建默认存档「**主账号**」。
- 顶部下拉菜单可切换存档；点击「**+ 新建**」输入名称创建新存档。
- 点击「**删除**」可删除当前存档（至少保留一个存档，唯一存档无法被删除）。

### 随机池

1. 在输入框中填写精灵名称（可选）。
2. 点击 **「+ 增加」** 记录一次随机捕捉。
3. 误操作可点击 **「- 减少」** 回退一次计数。
4. 获得异色后点击 **「↺ 重置」** 将计数归零。

### 家族池

1. 在输入框中填写精灵名称后点击 **「+ 添加家族」** 创建追踪条目。
2. 点击列表中某一精灵行，使用右侧的增 / 减 / 重置 / 删除按钮操作。
3. 顶部搜索框可实时过滤精灵名称。

### 属性池

1. 点击网格中的属性行将其选中（高亮显示）。
2. 使用下方的 **「+」 / 「−」 / 「↺」** 按钮操作选中属性的计数。
3. 复合属性精灵（如火 + 萌）需分别对两个属性各操作一次。

### 保底预警规则

| 计数范围 | 颜色 | 含义 |
|----------|------|------|
| 0 – 69 | 白色 | 正常 |
| 70 – 79 | 🟠 橙色 | 接近保底，注意 |
| 80 | 🔴 红色 | 已达保底上限 |

## 存档数据格式

存档以 JSON 格式保存于 `saves/<存档名>.json`，结构示意：

```jsonc
{
  "name": "主账号",
  "random_pool": 12,
  "family_pool": {
    "No.041 奇丽草": 5,
    "No.131 恶魔狼": 21,
    // ... 其余精灵
  },
  "element_pool": {
    "火": 8,
    "水": 3,
    // ... 其余 16 种属性
  },
  "global_count": 35,
  "logs": [
    {
      "timestamp": "2077-07-07 07:07:07",
      "pool_type": "random",
      "action": "increase",
      "target": "柴渣虫",
      "count_after": 1
    }
    // ...
  ]
}
```
